### ✨ 重磅新功能

#### 🔌 插件系统
全新的插件商店上线！现在你可以：

- 浏览并激活各种增强功能插件，让Mods Locker更强大

以前只能整个Mod启用或禁用，现在你可以：

- 查看Mod包内的详细文件结构
- 单独控制某个文件或文件夹的启用/禁用状态
- 一键重置所有文件设置，快速恢复默认
- 实时查看Mod文件的变更情况

![文件结构管理截图](https://pic.x-yue.top/i/2025/10/10/j9hpa.png)

专门为NBA 2K26 1.4版本升级后遇到兼容性问题的玩家准备：

- 一键修复IFF文件兼容性问题
- 支持批量处理所有受影响的Mod
- 智能检测，只修复需要的文件

![IFF Fixer截图](https://pic.x-yue.top/i/2025/10/10/j9k5r.png)

#### 📦 批量导入增强

- 支持一次性扫描多个Mod文件夹
- 实时显示导入进度和状态
- 导入后不再误报所有Mod需要更新

#### 🎯 分类管理升级

- **拖拽排序**：现在可以自由调整分类的显示顺序了
- 在Mod列表中可以批量分配分类和子分类
- 新增分类搜索功能，快速找到目标分类

#### 🎮 游戏快速启动
在管理页面新增"启动Modium"按钮，一键启动NBA 2K游戏

#### 📏 Mod大小显示
Mod信息页面现在会显示每个Mod包的文件大小，方便你了解存储占用

### 💡 使用提示

1. **首次使用插件系统**：进入"商店"页面查看可用插件，部分插件支持免费试用
2. **文件级别管理**：在Mod信息页面，点击"查看文件结构"按钮即可打开文件管理界面
3. **IFF Fixer**：如果你的2K26升级到1.4后Mod出现问题，在Mod信息页面点击"修复IFF文件"
4. **分类排序**：在管理页面，直接拖拽分类卡片即可调整顺序
5. **批量导入**：在Mod信息页面点击"扫描导入"，可一次性导入多个Mod文件夹
