## ⚡ パフォーマンスベンチマーク

単一の大型パッチ：約108 GB / 6ファイル → **約5秒**で切り替え  
全時代ジャージーパック：約18 GB / 3000ファイル → **約5秒**で切り替え  
結果：「応答なし」の待ち時間を大幅短縮 — ゲームに素早く戻れる

## 🔗 Modium統合（ベータ版）

**Modium統合**を開く → **Modiumを起動** → **ライブラリ / NBA2K25(26)**に移動  
Modium内でModiumのmodsを管理し、Mods Lockerが**展開/切り替え**を処理
