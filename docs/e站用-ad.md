# 🏀 Mods Locker — 方便、智能的 NBA 2K Mods 管理器

## 让 MOD 管理更简单、更直观、更安心

![简单图床 - EasyImage](https://pic.x-yue.top/i/2025/10/10/yrnia5.png)

### 🌟 为什么选择 Mods Locker？
- **零冗余部署**：链接技术不复制到游戏目录，节省磁盘、减少风险
- **瞬时启用/禁用**：一键开关，支持一键部署与一键清理
- **所见即所得**：卡片化 + 分类视图，定位内容更快
- **跨语言**：内置中 / 英 / 日界面
- **稳定可靠**：本地存储、增量更新、低资源占用

![简单图床 - EasyImage](https://pic.x-yue.top/i/2025/10/10/zewt4m.png)

## ✅ 已实装核心功能

### 📦 Mods 库与导入
- 扫描多个文件夹，批量导入 Mods
- 导入前可预览与批量重命名，避免误导入
- Mod 卡片支持自定义封面与备注，显示包体大小

![简单图床 - EasyImage](https://pic.x-yue.top/i/2025/10/10/zewwdw.png)

### 🗂️ 文件级别控制
- 查看 Mod 包完整文件树
- 对单个文件 / 文件夹做启用 / 禁用控制
- 一键重置；实时查看变更

![简单图床 - EasyImage](https://pic.x-yue.top/i/2025/10/10/zewx2c.png)

### 🏷️ 分类管理
- 自定义分类与子分类
- 支持排序、批量分配与分类搜索


### 🚀 启用与部署
- 开关控制 Mods 启用状态
- 一键部署至游戏目录
- 一键清理部署链接，环境保持干净

![简单图床 - EasyImage](https://pic.x-yue.top/i/2025/10/10/zewwds.png)

### 🎮 游戏快速启动
- 与 **Modium** 配合：一键启动游戏

### 🧰 IFF Fixer（2K26 1.4 兼容，感谢 L 大工具）
- 一键修复受影响的 IFF 文件
- 批量处理与智能检测，仅修必要文件


### 🧩 插件系统（插件商店）
- 浏览并激活增强插件，支持试用 / 永久激活
- 部分高级能力通过插件解锁
